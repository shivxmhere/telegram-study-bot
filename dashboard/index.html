How StudyBot Works

1. Send a Command

Open Telegram, find @studyiqbot, and send /study [any topic]

2. Wikipedia + NLP

Bot fetches Wikipedia, extracts key sentences, generates MCQ from content

3. Track Progress

Every correct answer adds to your score. Build daily streaks. Review history anytime.

Run the Bot Yourself

# Clone the repo
git clone github.com/shivxmhere/telegram-study-bot
cd telegram-study-bot/bot

# Install dependencies
pip install -r requirements.txt

# Add your bot token
cp .env.example .env
# Edit .env: BOT_TOKEN=your_token_here

# Run the bot
python bot.py
